# Contributing to Sound Radar Classifier

We welcome contributions! Here's how you can help:

## Ways to Contribute

- **Report Issues** - Found a bug or have a feature request? Open an issue.
- **Pull Requests** - Submit fixes or new features via pull requests.
- **Documentation** - Improve docs or add translations.

## Guidelines

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Getting Help

- Check the [documentation](docs/index.md) for setup and architecture details
- Review existing issues before opening a new one
